## 安装 Jupyter Notebook

```bash
pip3 install jupyter
```

## 添加 Node 支持

```bash
brew install zeromq
pip3 install --upgrade pyzmq
yarn global add ijavascript
ijsinstall
```

## 设置主题

```bash
pip3 install jupyterthemes
jt -t onedork
```

## 启动

```bash
ijsnotebook --port=8889
```
